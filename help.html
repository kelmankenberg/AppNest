Welcome to AppNest

AppNest is an application launcher designed to work in harmony with the Windows Start menu or as a standalone launcher. Its goal is to provide a seamless, efficient way to launch both installed and portable applications, enhancing workflow and organization for all types of users.

Key Features

  • Unified Launcher: Access both portable and installed applications from one interface
  • Organization Tools: Sort apps by favorites, categories, usage, and more
  • Quick Search: Find your applications instantly with the search feature
  • Folder Access: Quick access to important folders for seamless workflow
  • Drive Space Monitoring: Visual indicators of available disk space
  • Dark & Light Themes: Work comfortably in any lighting conditions

Getting Started

To get the most out of AppNest, we recommend starting with these steps:

  1. Add your most frequently used applications using the button in the Apps menu
  2. Create categories to organize your applications
  3. Mark your essential apps as favorites for quick access
  4. Experiment with different sort options to find your preferred view
  5. Customize AppNest's appearance in the Settings

General Settings

General settings control the basic behavior and functionality of AppNest.

Startup Options

Configure how AppNest behaves when your computer starts.

Start with Windows

When enabled, AppNest will automatically launch when you start your computer.

  • ON: AppNest starts automatically with Windows
  • OFF: AppNest must be started manually

Search Options

Control how the search functionality operates.

Search Mode

Choose which properties of your applications are included when searching.

  • Name Only: Search only matches application names
  • Name & Description: Search includes both names and descriptions
  • All Properties: Search includes all application properties (name, description, path, etc.)

Appearance Settings

Customize the visual aspects of AppNest to match your preferences and improve usability.

Theme

AppNest offers light and dark themes to match your preference and reduce eye strain.

Theme Selection

  • Light Theme: Brighter interface ideal for well-lit environments
  • Dark Theme: Darker interface that reduces eye strain in low-light conditions

Tip: The theme switcher is also available in the main window's footer for quick access.

Font Size

Adjust the text size in the application menu for better readability.

App Menu Font Size

Select the font size (in pixels) that works best for your display and preferences. The default is 14px.

  • Smaller sizes (12px): Fit more content on screen
  • Larger sizes (16-18px): Improve readability

Note: Font size changes apply to the application list only, not to the entire interface.

App Configuration

Configure how applications are displayed and organized in AppNest.

Application Display

Control how applications are presented in the AppNest interface.

Default Sort Method

Choose the default order for displaying your applications:

  • Alphabetical: Sort by application name (A-Z)
  • Most Used: Sort by frequency of use
  • Recently Added: Show newest additions first
  • Recently Used: Show recently launched applications first
  • Category: Group applications by their assigned categories

Show Icon Labels

Toggle whether application names are displayed beneath their icons.

  • ON: Display application names
  • OFF: Show icons only

Categories

Customize how application categories are managed and displayed.

Default Category

Set the default category assigned to newly added applications when no category is specified.

Show Empty Categories

  • ON: Display all categories, even those with no applications
  • OFF: Hide categories with no assigned applications

Folders Management

Configure how folders are displayed and managed in AppNest.

Folder Button Visibility

Control which folder types are shown in the folder navigation panel.

App Folders

These are folders defined specifically for AppNest, typically containing your portable applications.

  • ON: Show app folders in the navigation panel
  • OFF: Hide app folders

User Folders

Standard Windows user folders like Documents, Pictures, Downloads, etc.

  • ON: Show user folders in the navigation panel
  • OFF: Hide user folders

Individual Folder Visibility

For each app folder and user folder, you can toggle its visibility independently.

App Folders Configuration

App Folders Path

Set the base directory where AppNest looks for application folders. This is particularly useful for portable app organization.

Default: ./AppData (relative to the AppNest installation folder)

Changing the App Folders Path

  1. Click the "Browse" button next to the path field
  2. Navigate to your desired folder location
  3. Click "Select Folder" to confirm
  4. Click "Save" to apply the change

Warning: Changing the app folders path may require you to update your application paths if they were previously relative to the old location.

Resetting Settings

If you need to restore AppNest to its default configuration, you can reset all settings at once.

Reset to Default Settings

This function will restore all settings to their factory defaults. This can be helpful if:

  • You've made configuration changes that are causing issues
  • You want to start fresh with the original settings
  • You're troubleshooting problems with the application

Default Values

When reset, the following default values will be applied:

  • Theme: Light
  • App Menu Font Size: 14px
  • Folder Button Visibility: App Folders
  • All folder visibility toggles: ON (for both App Folders and User Folders)
  • App Folders Path: './AppData'
  • Start with Windows: OFF
  • Search Mode: Name Only

How to Reset Settings

  1. Open the Settings window
  2. Scroll to the bottom of any settings tab
  3. Click "Reset to Default Settings"
  4. Confirm your choice in the dialog that appears

Caution: This action cannot be undone. All your current settings will be permanently lost.
